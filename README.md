# Assistente de Voz Multi-idiomas (Gemini + Whisper) 🎙️🤖

Este projeto consiste em um assistente de voz capaz de ouvir, processar e responder de forma inteligente em diversos idiomas, utilizando tecnologias de Inteligência Artificial.

---

## 🛠️ Tecnologias Utilizadas
* **Whisper (OpenAI):** Reconhecimento de Fala (STT) executado localmente.
* **Google Gemini API:** Modelo de linguagem para processamento e geração de respostas.
* **gTTS (Google Text-to-Speech):** Síntese de voz para a saída de áudio.
* **JavaScript, Python & Google Colab:** Ambiente de desenvolvimento.

---

## 🚀 Funcionalidades
* Captura de áudio via interface JavaScript no navegador.
* Transcrição robusta a ruídos e sotaques com o modelo Whisper.
* Integração segura com API via Secrets do Colab.
* Resposta sintetizada em áudio.
